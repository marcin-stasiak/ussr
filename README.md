# USSR: Universal Server Side Renderer

## Description

It's basically **Universal Server Side Renderer** but written specifically for **Magento PWA**.

## Installation

```bash
$ npm install
```

## Configuration
```bash
# copy and edit .env file
$ cp .env.dist .env
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
